# 🔐 Генератор и анализатор паролей

Программа на языке C для генерации паролей (числовые последовательности и даты) и проверки их надёжности.

## 📋 Функциональность

1. **Генерация числовых паролей** — создаёт файл со всеми числами от 0 до 10ⁿ (с ведущими нулями)
2. **Генерация паролей из дат** — создаёт файл с датами в формате ДД.ММ.ГГГГ (от 01.01.0000 до 31.12.2026)
3. **Проверка надёжности пароля** — анализирует введённый пароль по следующим критериям:
   - Наличие цифр
   - Наличие строчных букв (a-z)
   - Наличие заглавных букв (A-Z)
   - Наличие специальных символов
   - Минимальная длина 8 символов

Пароль считается **надёжным**, если его длина ≥ 8 символов **И** выполнены как минимум 3 из 4 критериев.

## 🛠️ Требования

- Компилятор GCC
- Библиотека math.h (для функции pow)

## ⚙️ Компиляция и запуск

```bash
# Компиляция
gcc main.c create_passwords.c -lm -o password_manager

# Запуск
./password_manager
```

### Пояснение флагов:
- `-lm` — линковка с математической библиотекой (необходима для `pow()`)

## 🎮 Использование

После запуска программы отображается меню:

```
====================================
          Оператор паролей
====================================
            Меню
1 - создать пароль из чисел какой-то длины
2 - создать пароль из дат
3 - проверить пароль на надёжность
4 - показать меню
5 - выйти из программы
```

### Примеры:

**1. Генерация числовых паролей**

```
Введите количество чисел в пароле: 3
Сохранено в файл digit_password_3.txt
```

Файл `digit_password_3.txt` будет содержать:
```
000
001
002
...
999
```

**2. Генерация паролей из дат**

```
Готово
```

Файл `input_data.txt` будет содержать даты:
```
01.01.2026
02.01.2026
...
31.12.0000
```

**3. Проверка пароля**

```
Введите ваш пароль: Pass1234
Пароль надёжный
```

```
Введите ваш пароль: 1234567
Пароль не надёжный
```

## 🐛 Известные ограничения

1. **password_digit**: максимальная длина — 9 цифр (из-за ограничения типа `long int`)
2. **password_data**: не учитывает високосные годы (февраль всегда 29 дней)
3. **Ввод пароля**: максимальная длина — 99 символов
